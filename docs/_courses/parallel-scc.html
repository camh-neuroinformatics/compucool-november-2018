Software to connect to the cluster is also provided." --- The CAMH Compute Cluster is a core resource available to all researchers at CAMH. The Parallel Computing Workshop deals with leveraging multiple processing cores on a single machine, and scaling analyses throughout a compute cluster. Tools reviewed include GNU parallel and ‘qbatch’.
